# PPT 截图清单

本目录用于保存 PPT 和实验报告中引用的系统截图。当前不强制提交截图文件，建议最终数据库确定后统一截取。

建议文件名：

```text
01_dashboard.png
02_event_query.png
03_bilateral_relation.png
04_cooperation_analysis.png
05_risk_assessment.png
06_thematic_map.png
07_export_result.png
```

截图要求：

- 使用最终导入数据库，避免空页面。
- 截图前等待页面数据加载完成。
- 优先截取指标卡片、图表、表格和研判文本同时可见的区域。
- PPT 中只展示结果，不需要写现场操作步骤。
