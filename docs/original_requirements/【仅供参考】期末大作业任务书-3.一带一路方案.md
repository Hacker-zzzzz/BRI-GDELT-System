## 一、项目背景

"一带一路"倡议（Belt and Road Initiative, BRI）是新时代中国推动构建人类命运共同体的重要实践，涵盖亚洲、欧洲、非洲等 150+ 沿线国家的经贸、基建、人文合作。在国际形势日益复杂的背景下，如何**量化评估沿线国家之间的合作态势、监测潜在风险、识别合作机遇**，成为大数据管理与应用的重要课题。

GDELT 2.0 Event Database 记录了全球各国之间的每日交互事件，包含**合作/冲突行为编码、情感语调、地理信息、媒体报道量**等关键字段。通过筛选一带一路沿线国家相关事件，可以量化分析：
- 中国与某沿线国家的关系变化趋势
- 沿线国家之间的经贸合作活跃度
- 区域冲突风险与预警信号
- 不同子区域（东南亚/中亚/中东/东欧/非洲）的合作态势差异

本项目要求每组学生基于 GDELT 2.0 Event 数据，开发一套聚焦一带一路沿线国家的**合作态势分析、风险评估、专题地图展示**的 Java 桌面应用软件。

## 二、项目目标

### 2.1 知识目标
- 掌握 Java 面向对象编程、集合框架、I/O 操作、多线程编程
- 掌握 Java GUI 开发技术（JavaFX 或 Swing）
- 掌握嵌入式数据库（SQLite/H2）的 JDBC 操作
- 了解 GDELT 事件数据结构与 CAMEO 编码体系
- 了解一带一路倡议背景与国际合作量化分析方法
- 掌握综合评价模型构建方法（指标体系、权重计算）

### 2.2 能力目标
- 能独立完成大型 TSV 数据的解析、筛选、入库
- 能设计并实现多模块协同的桌面应用系统
- 能运用 Weka 等 Java 库完成数据分析与聚类任务
- 能用专题地图和仪表盘展示分析结果
- 能通过团队协作完成项目开发与文档撰写

### 2.3 素养目标
- 培养大数据思维：从海量事件中提取有价值的合作信号
- 培养国际视野：理解"一带一路"国际合作格局
- 培养家国情怀与数据科学社会责任感
- 培养团队协作与项目管理能力

## 三、功能需求

### 3.1 数据管理模块

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| F-01 | 数据导入 | 支持导入 GDELT 2.0 TSV 格式数据文件，支持指定国家代码过滤导入（仅导入一带一路沿线国家相关记录），进度条显示导入进度 | **必做** |
| F-02 | 数据存储 | 将筛选后的数据存入嵌入式数据库，设计合理表结构，包含事件主表和国家元数据表 | **必做** |
| F-03 | 数据清理 | 空值处理、异常经纬度过滤、合作/冲突事件分类标记 | **必做** |
| F-04 | 数据导出 | 支持将分析结果导出为 CSV/Excel 格式 | **必做** |
| F-05 | 沿线国家配置 | 内置一带一路沿线 150+ 国家列表（JSON 配置），支持按子区域（东南亚/中亚/中东/东欧/非洲）筛选 | **必做** |

### 3.2 查询检索模块

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| F-06 | 基础查询 | 按日期范围、国家（双边）、事件类型（合作/冲突）、子区域进行组合条件查询 | **必做** |
| F-07 | 双边关系查询 | 输入中国 + 某沿线国家，查询两国之间所有交互事件，按时间排序展示 | **必做** |
| F-08 | 子区域查询 | 选择某一子区域（如"东南亚"），查询该区域内所有国家间的事件汇总 | **必做** |
| F-09 | 合作事件筛选 | 按 CAMEO 合作类事件编码（04-06类）筛选，展示经济合作、外交合作、援助等子类分布 | **必做** |
| F-10 | 冲突事件筛选 | 按 CAMEO 冲突类事件编码（08-14类）筛选，展示制裁、威胁、军事冲突等子类分布 | **必做** |
| F-11 | 模糊搜索 | 按国家中文名称/英文名称模糊搜索，自动补全 | 选做 |

### 3.3 挖掘分析模块

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| F-12 | 合作指数计算 | 构建综合合作指数 = w1×合作事件量 + w2×Goldstein均值 + w3×AvgTone + w4×媒体关注度，对沿线国家进行排名 | **必做** |
| F-13 | 关系趋势分析 | 计算中国与指定国家的合作指数月度趋势曲线，识别上升期/下降期/拐点 | **必做** |
| F-14 | 区域聚类分析 | 基于合作指数、冲突频率、Goldstein 均值等特征，用 K-Means 将沿线国家分为"深度合作伙伴""稳定合作""存在风险""高度紧张"等类别 | **必做** |
| F-15 | 投资风险评估 | 综合冲突事件量、制裁频率、Goldstein 最低值、AvgTone 负面程度，计算各国投资/合作风险等级（低/中/高/极高） | **必做** |
| F-16 | 合作热点追踪 | 找出合作指数环比增长最快的 TOP10 国家（"新兴合作热点"） | 选做 |
| F-17 | 关联规则挖掘 | 使用 Apriori 挖掘"当 A 国与 B 国发生经贸合作后，后续倾向于发生 C 类事件"的关联规则 | 选做 |

### 3.4 可视化模块

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| F-18 | 合作态势总览仪表盘 | 首页仪表盘：沿线国家总数、合作事件总量、冲突事件总量、当前热点 TOP5、风险预警 TOP5 | **必做** |
| F-19 | 关系趋势折线图 | 展示中国与指定国家的合作指数月度趋势、Goldstein 趋势、情感趋势（多曲线叠加） | **必做** |
| F-20 | 合作指数排行榜 | 柱状图展示沿线国家合作指数排名（分区域着色） | **必做** |
| F-21 | 一带一路专题地图 | 在世界地图（或亚欧非地图）上渲染沿线国家，颜色=合作等级（绿→黄→红），气泡大小=合作事件量 | **必做** |
| F-22 | 区域对比雷达图 | 五大子区域（东南亚/中亚/中东/东欧/非洲）的多指标雷达图对比 | **必做** |
| F-23 | 事件类型饼图 | 合作/冲突事件子类分布饼图或堆叠柱状图 | 选做 |
| F-24 | 年度报告生成 | 自动生成指定国家的年度合作报告（文本描述 + 图表），导出为 PDF/HTML | 选做 |

## 十一、数据指南

### 11.1 数据下载
```
http://data.gdeltproject.org/gdeltv2/YYYYMMDD.export.CSV.zip
```

### 11.2 文件格式
- **分隔符**：Tab（`\t`），不是逗号
- **编码**：UTF-8，无表头行
- **压缩**：`.zip`

### 11.3 关键字段
- 字段 6/16：Actor1/2 国家代码（CAMEO 三位码）
- 字段 27/28/29：EventCode / EventBaseCode / EventRootCode
- 字段 31：GoldsteinScale（合作-冲突分值）
- 字段 35：AvgTone（媒体情感语调）
