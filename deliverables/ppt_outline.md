# 一带一路沿线国家合作态势分析系统 PPT 大纲

## 1. 封面

- 题目：一带一路沿线国家合作态势分析系统
- 课程：Java 程序设计期末大作业
- 组员姓名、学号、分工

## 2. 选题背景

- 一带一路沿线国家互动频繁，合作与风险并存。
- GDELT 提供全球事件数据，可用于量化国际关系变化。
- 本项目使用 Java 桌面程序实现数据导入、分析和结果展示。

## 3. 需求分析

- 数据导入与清洗
- 数据库存储
- 多条件事件查询
- 双边关系分析
- 合作态势分析
- 风险评估
- 图表和专题地图
- 结果导出

## 4. 技术路线

- Java 17：核心开发语言
- JavaFX：桌面窗口与图表界面
- SQLite：本地关系数据库
- Maven：依赖管理和运行
- GDELT 2.0 Event：数据来源

## 9. 合作指数

口径说明：

- 合作事件数量越高，指数越高。
- GoldsteinScale 越正，说明合作倾向越强。
- AvgTone 越正，媒体语调越积极。
- NumMentions 表示媒体关注度。
- 冲突事件会对合作指数扣分。

## 10. 风险指数

口径说明：

- 冲突事件数量和冲突占比越高，风险越高。
- GoldsteinScale 越负，风险越高。
- AvgTone 越负，媒体语调越消极。
- 风险等级分为低、中、高、极高。
